Supply Chain System Policies

Policy Description
Supply Chain Default Active Supply Chain ID The ID of the default supply chain
Supply Chain Garbage Collection Max Minutes Allowed for Processing The max number of minutes that garbage collection can be performed. Default is 60 minutes.
Supply Chain Garbage Collection Max Number of Deleted Supply Chain to Process at a Time The max number of deleted supply chains to process at one time. Default is 100 supply chains.
Supply Chain Garbage Collection Minutes to Wait Before Removal of Deleted Entries The max number of minutes JustEnough waits after a supply chain is deleted to remove the entry plus related entries from the system. Default is 129600 minutes (90 days).
Supply Chain Override With From Supplier Flag When set to True, the Supply Chain Hierarchy will be ignored for Inventory that have a From Supplier Flag set to true; however, Supply Chain Overrides will always be used if defined.
