Editing Permission Sets

  1. In the Workflow Manager screen, select the workflow containing the permissions you want to edit.
  2. In the Design pane, select the status whose permission set you want to edit.

  3. In the Status Permission blacklist, select the permissions you want to edit and click Edit.

    In this example, we've selected Edit Price Action and Delete Price Action user rights.

  4. In the Remove User Rights dialog box:

    1. Select the user right/status state combinations you want to customize permissions for.

      In this example, we only want to customize edit and delete rights for "valid" price actions.

    2. Choose the roles/users you want to remove rights for in those circumstances:

      In this example, only users with Administrator or JustEnough Advanced User roles will be able to edit or delete valid price actions while the price action is in the Ready for Approval status.

    3. Click Save. The Status Permission Blacklist is updated to reflect the changes you made.

      In this example, the blue user icons in the Valid column indicate that Edit Price Action and Delete Price Action rights will be removed for one or more users when the status is in the Valid state.

Note: Alternatively, you can click on a user icon to view/edit permissions for that specific user right/status state combination.
