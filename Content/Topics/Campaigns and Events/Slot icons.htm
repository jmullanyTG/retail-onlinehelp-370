Slot icons

Slots may have an icon displayed in the top right corner that indicates how the Slot is assigned.

The icons signify the following:

Icon

Description

Pages

Indicates that the Slot is fully assigned with multiple Promotions.

Exclamation

Indicates that the Slot is partially assigned. One or more Geographies are available.

No Icon

If there is a Promotion assigned to the Slot, no icon in the top right corner would indicate that the Promotion is assigned to all Geographies. If no Promotion is assigned to the Slot, it indicates that the Slot is fully open.
