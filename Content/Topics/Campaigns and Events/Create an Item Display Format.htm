Create an Item Display Format

In the Item Display Formats tab, click New to create a new Item Display Format, enter the required fields and click Ok.

 

Field

Description

Name (Required)

Enter a descriptive label to summarize the purpose of the Item Display Format. This will be used to identify the Item Display Format when setting up Slot Set Templates or modifying the Layout of an Event Slot Set.

Units (Required)

Modify the default unit of measure from the dropdown list.

Width (Required)

Modify the default width of the Item Display Format box.

Height (Required)

Modify the height of the Item Display Format box.

 

The following buttons and actions may also be helpful when creating an Item Display Format:

Action

Description

Delete

Select the field from the Item Display Format box on the right and then click on this button.

Clear all

Remove all sizing, font attributes, text and attribute fields from the Item Display Format.

Dock Layout Control

Align the containers as needed in the Item Display Format.

Zoom slider

Used to adjust the display size of the Item Display Format while viewing in Configuration.

 

Content Fields, Images, Price Fields and Attribute Display Formats are added to the Item Display Format by using Add Field, Add Image, Add Price and Add Attribute Display. Click Ok to save the changes or Cancel to revert to the previously saved version of the Item Display Format.
