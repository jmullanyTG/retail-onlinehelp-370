Add price

Add Price will add a Price to the Item Display Format. The Price can be resized and moved in the box on the right and the following display settings can be modified as needed. Please note that these attributes are used to display a Price placeholder when the Item Display Format is viewed without a Promotion associated to it. Once a Promotion is associated with an Item Display Format, the appropriate Price Display Format configuration is used to display Price.

Field

Description

Font (Optional)

Modify the font used to display the Price.

Note: Used for Item Display Format previewing purposes only. This configuration will not affect the actual layout on the Event Slot.

Size (Optional)

Modify the font size of the text in the Price.

Note: Used for Item Display Format previewing purposes only. This configuration will not affect the actual layout on the Event Slot.

Color (Optional)

Modify the font color of the text in the Price.

Note: Used for Item Display Format previewing purposes only. This configuration will not affect the actual layout on the Event Slot.
