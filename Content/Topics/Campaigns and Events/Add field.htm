Add field

Add Field will add Content Fields to the Item Display Format. The field may be resized and moved in the box on the right and the following additional attributes can be modified as needed.

Field

Description

Field (Optional)

Select a Content Field from the dropdown list to display in the Item Display Format.

Font (Required)

Modify the font used to display text in the field.

Size (Required)

Modify the font size of the text in the field.

Color (Required)

Modify the font color of the text in the field.
