Add attribute display

Add Attribute Display will add Attribute Display Formats to the Item Display Format. The field can be resized and moved in the box on the right and the following display settings can be modified as needed.

Field

Description

Attribute Display (Optional)

Select an Attribute Display Format from the dropdown list to display in the Item Display Format.

Font (Optional)

Modify the font used to display text in the Attribute Display Format.

Note: Used for Item Display Format previewing purposes only. This configuration will not affect the actual layout on the Event Slot.

Size (Optional)

Modify the font size of the text in the Attribute Display Format.

Note: Used for Item Display Format previewing purposes only. This configuration will not affect the actual layout on the Event Slot.

Color (Optional)

Modify the font color of the text in the Attribute Display Format.

Note: Used for Item Display Format previewing purposes only. This configuration will not affect the actual layout on the Event Slot.
