Create an Attribute Display Format

In the Attribute Display Formats tab, click New to create a new Attribute Display Format, enter the required fields and click Ok.

Field

Description

Name (Required)

Enter a descriptive label to summarize the purpose of the Attribute Display Format. This will identify the Attribute Display Format when setting up an Item Display Format.

Units (Required)

Modify the default unit of measure from the dropdown list.

Width (Required)

Modify the default width of the Attribute Display Format box. The size of the box within Item Display Formats should be taken into consideration when determining the appropriate dimensions of the Price Display Format.

Height (Required)

Modify the height of the Attribute Display Format box. The size of the box within Item Display Formats should be taken into consideration when determining the appropriate dimensions of the Price Display Format. Click on Add Text to add text to the Attribute Display Format. Modify the text fields below and update default text in the Text Preview box.

Font (Required)

Modify the font used to display text in the Attribute Display Format.

Size (Required)

Modify the font size of the text in the Attribute Display Format.

Color (Required)

Modify the font color of the text in the Attribute Display Format.

Attribute (Optional)

Select a customer specific attribute field from the dropdown list to display in the Attribute Display Format and click on the down arrow icon to append the field below the existing fields in the Attribute Display Format.

 

 

The following buttons and actions may also be helpful when modifying an Attribute Display Format:

Action

Description

Delete

Select the field from the Attribute Display Format box on the right and then click on this button.

Clear all

Remove all sizing, font attributes, text and attribute fields from the Attribute Display Format.

Zoom slider

Used to adjust the display size of the Attribute Display Format while viewing in Configuration.
