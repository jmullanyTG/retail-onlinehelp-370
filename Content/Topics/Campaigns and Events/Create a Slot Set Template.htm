Create a Slot Set Template

In the Slot Set Templates tab, click New to create a new Slot Set Template, enter the required fields and click Ok.

Field

Description

Name (Required)

Enter a descriptive label to summarize the purpose of the Slot Set Template. This is used to identify the Slot Set Template when setting up an Event Type or Event Slot Set.

Units (Required)

Modify the default unit of measure from the dropdown list.

Width (Required)

Modify the width of the Slot Set Template.

Height (Required)

Modify the height of the Slot Set Template.

Rows (Required)

Select the number of rows to add to the Slot Set Template. This value combined with the Columns value will generate a grid of Slots in the Slot Set Template.

Columns (Required)

Select the number of columns to add to the Slot Set Template. This value combined with the Rows value will generate a grid of Slots in the Slot Set Template.

Default Space (Required)

Select the default Space to assign all Slots to in the new Slot Set Template.

Item Display Format (Required)

Select the default Item Display Format of the Slots in the new Slot Set Template.

 

After the new Slot Set Template has been created, the Edit Slot Set Template screen will be displayed and you can modify the new Slot Set Template as needed. For more information, see Modify a Slot Set Template.
