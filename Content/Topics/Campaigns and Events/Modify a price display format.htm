Modify a price display format

To modify a Price Display Format, select the Price Display Format and click on Edit. Make the necessary changes and click on Ok.

Field

Description

Name (Read Only)

The name of the Price Display Format is the same as the associated Promotion Strategy.

Units (Required)

Modify the unit of measure from the dropdown list.

Width (Required)

Modify the width of the Price Display Format box. The size of the box within Item Display Formats should be taken into consideration when determining the appropriate dimensions of the Price Display Format.

Height (Required)

Modify the height of the Price Display Format box. The size of the box within Item Display Formats should be taken into consideration when determining the appropriate dimensions of the Price Display Format.

Font (Required)

Modify the font used to display Price Fields in the Price Display Format.

Size (Required)

Modify the font size of the Price Fields in the Price Display Format.

Color (Required)

Modify the font color of the Price Fields in the Price Display Format.

Order Number (Optional)

An Order Number can be set up for each container in a Price Display Format. The Order Number is used to determine what order to display the Price Format container when populating reports. Reports are generally configured by Professional Services and should not be modified without assistance from JustEnough Professional Services or Support.

Price Field (Optional)

To add a Price Field, select it from the dropdown list and click on the down arrow icon to append the field to the container currently selected in the Price Display Format.

 

The following buttons and actions may also be helpful when modifying a Price Display Format:

Action

Description

Add Text

Add a new text container to the Price Display Format. Modify the default text in the Text Preview area.

Dock Layout Control

Align the containers as needed in the Price Display Format.

Delete

Select the container to remove from the Price Display Format box on the right and then click on this button.

Clear All

Remove all sizing, font attributes, text and price fields from the Price Display Format.

Zoom slider

Used to adjust the display size of the Price Display Format while viewing in Configuration
