Layout edit

The Layout view, is the only place where the user has the option to customize the Event Slot Set design, including Slots, their Positions, Space assignment and Item Display Format assignment. By selecting the blue pencil icon , the Slot Set transforms into a Layout Edit mode where a number of actions can be performed. Certain actions may be restricted based on user rights.

The following buttons may be helpful when editing a Slot Set:

Action

Description

Save

Save changes periodically while working on the Slot Set Template to make it easier to revert to a recent version of the Template. Once the Slot Set Template is saved, the Undo button is reset so Slot Set Template cannot be reverted to a state prior to saving.

Slot Split: Multiple

Split the selected Slot into multiple Slots. The new Slots are added to the right and below the existing Slot. All of these Slots are sized to fit into the same dimensions as the original Slot.

Slot Split: Horizontal

Horizontally split the Slot into two equally sized Slots one above the other. The user may drag and drop the border to adjust the relative sizes of the two newly split Slots.

Slot Split: Vertical

Vertically split the Slot into two equally sized Slots one beside the other. The user may drag and drop the border to adjust the relative sizes of the two newly split Slots.

Merge

Select the Slot and click on this button to merge it with its sibling Slot. The Merge Indicator should be enabled when performing this action to indicate which is the sibling Slot.

Clear all

To remove all Slots and their associated properties such as Item Format Display and Space, select this button.

Undo

Undo the most recent change(s) to the Slot Set Template. If the user has clicked Save at any time during modifications to the Slot Set Template, any changes made before saving cannot be undone.

Redo

If the Undo button was used, the most recent change(s) can be put back to the Slot Set Template by clicking on Redo. If the user has clicked Save at any time during modifications to the Slot Set Template, any changes made before saving cannot be put back.

Merge Indicator

Enabling the Merge Indicator will display the selected Slot’s sibling as faded. This will help you to know which Slot the current Slot will be merged with if you click on Merge.

Customization

If Customization is enabled and a Slot is selected, the Dock Layout Control is displayed and provides the ability to adjust the current Slot alignment and positioning in relation to the other Slots on the Slot Set. For more information, see Dock Layout Control.

Zoom slider

Used to adjust the display size of the Slot Set Template while viewing in Configuration.

In the top right Slot Information pane, the Slot details are displayed for the selected Slot.

In the bottom right Space Allocation pane, Space Allocation by Slot Set and the overall Event is displayed.

The Header and Slot Dimensions icons can be selected in conjunction with Layout Edit to help design the Slot Set.

When all changes are complete, click the Layout Edit icon again to exit out of design mode. Click Yes to save the changes to the Event.
