Configuring Demand Transference for Showroom Items

If you have items sold in showrooms (locations where the item is sold but not fulfilled):

  • Ensure that the Allowed Inventory Order Policies system policy includes the value “S” for showroom.
  • Configure an inventory policy for the item with the following settings:
    • Order Policy Type = Showroom
    • Minimum Safety Stock Units = # of items to display in showroom (for example, “1”)
    • Demand Transference:
      • Designated Fulfillment Site: To designate a single site as the fulfillment location for showroom sales, select this option and choose the fulfillment site.
      • Demand Transference Method: If showroom orders can be fulfilled from multiple locations, select this option and choose Split Factor.
