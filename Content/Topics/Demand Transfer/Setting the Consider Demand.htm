Setting the Consider Demand Transference Option for an Item

  1. From the Manager tab, click Item Manager.
  2. If the Items dialog box does not appear, click Inventory Manager Document on the toolbar to open it.
  3. In the Items dialog box, select the filter containing the items you want to manage.
  4. In the Choose Fields dialog box, add the Consider Demand Transference field.
  5. In each product row of the inventory manager document, select the Consider Demand Transference checkbox to toggle including/ignoring demand transfers when calculating replenishment for that item.
  6. Click Save to save any changes.
