Demand Forecasting

The following enhancements are now available in the Demand Forecasting module.

Shape Forecasting

A new forecasting method, Shape Forecasting, applies a pre-selected curve to an item's moving-average sales history to generate a moving-average based, season-affected demand forecast for that item.

The Shape Forecasting method appears in the Forecast task dialog box and participates in the forecast tournament when selected.

This method requires the following setup tasks.

  • Define appropriate curves in the Curves module.
  • Identify the desired curve for each item in the new Shape Forecast Maintenance view.

Shape Forecast Maintenance View

The Shape Forecasting Maintenance view is where you identify the curve that JustEnough should use to generate an item's shape forecast.

  1. In the Views module, open the Shape Forecast Maintenance view.

  2. In the Filter list, choose the filter containing your desired item(s). The items appear in two grids. The top grid lists items at sites. The bottom grid lists items by channel. (The same items appear in both grids.)

  3. To define an item's curve, select the item in a grid and click in its Curve cell to display the "..." button. Click the "..." button to open the Curve Selection dialog box.

  4. In the Curve Selection dialog box, choose the curve you want to apply by selecting its row in the grid and clicking OK.

  5. Click Save above the grid to save the change.

  6. Repeat steps 2-5 for all desired items.

    The next time that JustEnough generates demand forecasts for these items, it will use these curves to create their shape forecasts.

New Error Method: Absolute Total Error (ATE) BETA FEATURE

A new method for calculating method fit is now available as a beta feature. This method, Absolute Total Error (ATE), produces comparable-to-superior forecasts to the current error method, MAPE, in nearly all scenarios tested by JustEnough data scientists, including long-term products, high-sales products, low-sales products, intermittent-sales products, highly seasonal products, and products with strong trends.

Although this new method is recommend by JustEnough data scientists, caution is urged before using it in production environments. We encourage you to use ATE in your non-production environments first to understand its differences to and advantages over MAPE, and contact us if any issues arise.

To set ATE as the error method used by your JustEnough application, a new system policy, Forecast Error Method, has been added. Enter 'ATE' as the new policy value to start using it as the error method for all forecasts going forward. (As it is a system-wide policy, there is no way to use MAPE for some forecasts and ATE for others without manually switching the policy value each time.)

For reference, the Error Method property in a forecast's Details pane indicates which error method was used when generating the forecast.

For technical details related to ATE, including its calculation, comparison to MAPE, case studies, and examples, see Error Method Overview.

Method Fit column added to Item list

When selecting an item in the drop-down Item list, you can now see, sort, and filter by the method fit of the method chosen for the item's forecast.
