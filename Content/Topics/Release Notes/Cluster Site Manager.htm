Cluster Site Manager

The following enhancement is now available in the Cluster Site Manager module.

Multi-select option for snapshot and calculate cluster set actions

The Allow Multiple toolbar option has been extended to Snapshot and Calculate actions, making it much easier to create snapshots of multiple cluster sets at once and calculate multiple cluster sets at once.

When the Allow Multiple toolbar option is enabled, you can select multiple cluster sets and/or cluster set partitions in the grid and then click Delete, Snapshot, or Calculate to perform the action on all of the selected cluster sets.

Notes:

  • Selecting a cluster set partition automatically selects all of its cluster sets.
  • Multi-cluster set operations may be submitted as tasks to the workload queue. You will receive a notification once the operations are done.
