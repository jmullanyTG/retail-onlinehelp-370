System Policies

The following system policies have been added. See System Policies Overview for more information about system policies.

Engine: Forecasts

Display Name System Name Description
Forecast Error Method Forecast_ErrorMethod

The error method JustEnough uses to calculate a forecast's method fit. Options are "UniformMAPE" and "ATE".

Module: Allocation

Display Name System Name Description
Allocations Always Run Retrend Calculation Allocate.Always.Retrend

Whether or not to always calculate a retrend forecast when executing an allocation forecast.

True = JustEnough always runs the retrend regardless of the "Use Retrend" strategy line setting.

False = JustEnough does not run a retrend for a strategy line if its "Use Retrend" strategy line setting. is not selected.

Module: Events

Display Name System Name Description
Event Level Output Event.EventLevelOutput

Whether to output event slots sets as a single file or as individual files.

Event = all slot sets are output to a single file.

Slot Set = slots sets are output individually to their own files.

Module: Item Manager

Display Name System Name Description
Column Max InventoryManager.ColumnMax

The maximum number of columns allowed in a document. Applies to documents in Item Manager, Site Manager, Variant Manager, Supplier Manager, and Promotions modules.

Default value = 100,000.
Row Max InventoryManager.RowMax

The maximum number of rows allowed in a document. Applies to documents in Item Manager, Site Manager, Variant Manager, Supplier Manager, and Promotions modules.

Default value = 100.

 

Module: Price Planning

Display Name System Name Description
Pricing Plan View Collection Set PriceManager.ViewCollectionSetForPricingPlan

Sets whether or not to display a view collection set as a tab in the Price Manager's pricing plan screen.

Value = 0 does not display a view collection set tab.

Value = n (where "n" = a view collection set ID) displays the corresponding view collection set as a tab in a pricing plan screen. The tab name is the name of the selected view collection set.
