Promotion Manager

The following promotion-related enhancements are now available.

Configuration Migration utility

A new utility lets you migrate promotion-related configurations between QA, UAT, and production environments from within the JustEnough application. Previously, such migrations were done manually, which was a time-intensive and error-prone process. Now, what used to take days or weeks now takes only minutes to complete.

For more information, see Configuration Migration documentation in the online help.
