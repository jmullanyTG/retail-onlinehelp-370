General Enhancements

The following enhancements affect multiple JustEnough modules.

Maximum Row and Column Limit

Two new system policies (Module: Item Manager > Column Max and Row Max) limit the number of rows and columns that can be displayed in a JustEnough document. These policies apply to the following modules:

  • Item Manager
  • Site Manager
  • Supplier Manager
  • Variant Manager

For more information, see New System Policies.

New metadata field: Volume

A new metadata field, Volume, is available to item filters and Item Manager documents. This field was previously available in the database (tbl_ItemVolumeMetric) and is now available to the UI.
