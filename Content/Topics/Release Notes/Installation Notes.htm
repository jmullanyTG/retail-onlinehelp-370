Installation Notes

The upgrade process has changed for 3.7.0.

Refer to the Demand Management Installation Guide for detailed instructions and information related to upgrading from 3.6.2 to 3.7.0 as well as installing new instances of 3.7.0.

Minimum requirements

Software

.Net Framework 4.7.1 and Dot Net Core 2.1.3 are required on the Application Server and on Demand Management clients. These can be installed with the services or downloaded and installed from Microsoft.

Application Server

Supported operating systems:

  • Windows Server 2016 (recommended)
  • Windows Server 2012 R2
  • Windows Server 2008 R2 SP1 (not recommended)

Database Server

Supported operating systems:

  • Windows Server 2016 (recommended)
  • Windows Server 2012 R2

SQL Server:

  • SQL Server 2017 (recommended)
  • SQL Server 2016
  • SQL Server 2014 (not recommended)

Workstation

Supported operating systems:

  • Windows 10
  • Windows 8.1

Hardware

Application Server

CPU: Single Oct (8) Core CPU

RAM : 64 GB RAM

Windows 2016 Standard x64

HD: OS 300GB 15k SAS (RAID 1)

Database Server

CPU: Dual Quad core CPU

RAM :160GB RAM

Hard Disk Config: (Separate LUNS)

- OS - 100GB RAID 1 15K RPM

- DATA_1 - 400GB RAID 10 (SAS SSD / Separate LUN)

- DATA_2 - 400GB RAID 10 (SAS SSD / Separate LUN)

- LOG - 400GB RAID 1 (SSD / Separate LUN)

- TEMPDB - 1.6TB PCIe SSD

- ADMIN/BACKUP - 600GB RAID 1 15K RPM

Workstation

CPU: Single Quad (4) core CPU

RAM (Recommended: 6GB)

Hard Disk (Recommended: 1 GB *Free Space)

Graphics: Minimum 1024 X 768 Pixels (24 bit color), Recommended: 1920 X 1080 Pixels (24 bit color).
