Event Manager

The following enhancements are now available in the Event Manager module.

Improved UI for slot set template configuration

A new user interface handles memory better and helps prevent misconfiguration of slot set templates.

With this new interface, you can easily:

  • Renumber slots

  • Make adjacent slots the same height and/or width*

  • Merge slots and preserve the combined area perimeter*

  • Split slots into two or more equal-sized objects*

* Supported items include price display formats, attribute display formats, item display formats, slot set templates, and event slot sets.

NOTES:

  • This enhancement automatically updates existing price display formats, attribute display formats, item display formats, slot set templates, and event slot sets to be compatible with the new interface. This update occurs the first time you open Event Manager. The number of updates required determines how long the update process takes. DO NOT cancel this process or close the application before the update finishes.
  • Items with complex configurations may require additional manual updating. Customers upgrading from previous versions of JustEnough should verify these items in a JustEnough 3.7 QA\UAT environment before deploying JustEnough 3.7 in production systems.

Ability to generate all event slot sets as a single file

A new system policy, Event Level Output, sets whether an event's slot sets are output to a single file or to multiple files.
