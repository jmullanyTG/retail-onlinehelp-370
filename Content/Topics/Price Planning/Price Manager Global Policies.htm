Price Manager Global Policies

Global Policies are configured by Professional Services and shouldn’t be modified without assistance form JustEnough Professional Services or Support.

Global Policy

Description

Auto update Price Action on import changes

Controls the system to implement automatic updates for imported product changes. By default, the system is set to ‘False’ requiring you to review changes before updating the Price Action with the new information from the user interface.

If set to ‘True’ the system will automatically update Price Actions with imported changes that affect Pricing Actions with the same Product and Site combination.

Price Manager Config View Collection Set

This policy determines the user interface to be used for the Price Manager’s Configuration.

Price Manager Data Initialize Batch Size

This policy determines the number of records to return, at one time for performance reasons, when initializing a Price Action. The system default is 1000 Item Masters records. This will need to be modified based on the available server configuration.

Price Manager Inventory Meta Data View

This policy determines the default fields to be used when displaying Item Manager from Price Planning.

Price Manager Plan Start Date Number Of Days Out

This policy denotes the number of days in the future the system will configure a newly created Pricing Plan’s Start Date. This can only be overridden (i.e. backdated) with the Role/User Permission ‘Emergency Pricing’ which will allow a date to be selected as early as tomorrow.

Price Manager Price Action Calendar View Collection Set

This policy determines the user interface to be used for the Price Manager’s Calendar.
