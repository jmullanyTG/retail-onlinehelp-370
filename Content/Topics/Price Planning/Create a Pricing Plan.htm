Create a Pricing Plan

Within the Price Manager Home screen, click New to create a new Pricing Plan, enter the required fields and click Ok. Additional products and sites may be configured at a later time.

Field

Description

Pricing Plan Name (Required)

Enter a descriptive label to summarize the purpose of the Pricing Plan.

Pricing Plan Description (Optional)

Enter a summary of the Pricing Plan.

Pricing Plan Start Date (Required)

Select a future date that associated Pricing Actions will be defaulted to. The system will default the Start Date and restrict selection of an earlier date, by a ‘globally’ configured number of days forward from today. Special permissions (i.e. Emergency Pricing) will allow you to override this restriction and allow selection of tomorrow at the earliest.

Pricing Plan Variant Filter (Required)

Select a preconfigured filter form the drop down or alternatively click Maintain Filter to configure a new one to designate the products to be used within the Pricing Plan’s Action(s). This will be the initial set of products to be added to the Price Action. Additional Products may be added or removed at a later time.

Pricing Plan Site Filter (Required)

Select a preconfigured filter form the drop down or alternatively click Maintain Filter to configure a new one to designate the sites to be used within the Pricing Plan’s Action(s). This will be the initial set of Sites to be added to the Price Action. Additional Sites may be added or removed at a later time.
