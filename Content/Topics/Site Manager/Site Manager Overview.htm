Site modeling

When opening a new store or DC, you can choose an existing store or DC to model demand forecasts after. This creates a store supersession, which tells JustEnough to use the product-level sales history from the model site to forecast demand for the new site. Orders for the new site can then be generated based on the modeled forecasts.

Note: Store supersessions do not include forecasts based on item supersessions. To get demand forecasts based on item supersessions for the new site, simply edit the item supersession and link it to the new site.

Global priority
