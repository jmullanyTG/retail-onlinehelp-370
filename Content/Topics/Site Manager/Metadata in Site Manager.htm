Metadata in Site Manager

A wealth of metadata about your sites is available in Site Manager. You can add this data as columns in Site Manager documents and use it in site filters to find sites which match specific criteria.

Here are some examples of how metadata in Site Manager can help you better understand how sites are being used across JustEnough.

  • Filtering comparative and non-comparative sites.
  • Filtering sites in site clusters.
  • Filtering sites in location strategies.
  • Finding unconfigured supply chains.
